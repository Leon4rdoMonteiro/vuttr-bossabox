Documentação - VUTTR BossaBox V1

  • 🚀 API para cadastro de ferramentas mais utilizadas pela BossaBox. 🚀

Visão Geral ¶

Descrição ¶

  • A API desenvolvida tem por objetivo principal, armazenar as ferramentas mais utilizadas, sendo possível listá-las e pesquisá-las por tags de referência.

  • Possui também cadastro e autenticação e usuários, sendo obrigatório para cadastrar e gerenciar novas ferramentas na plataforma.

  • No momento da criação de uma nova ferramenta, é possível incluir um ícone que represente a mesma para melhor identificação visual.

Token de Autenticação ¶

  • Um token é necessário para autenticação de usuários na aplicação, o mesmo é gerado quando um usuário se cadastra na plataforma ou quando realiza log in.

    • Exemplo:
+ Authorization: Bearer f87gd87da76gf76da879adf897af89
